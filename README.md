# Learning Terraform

Hands-on **Terraform** exercises for **TKJ (Teknik Komputer & Jaringan)** students at **SMKN 1 Seyegan**.

This repository contains various Infrastructure as Code (IaC) projects to practice provisioning and managing infrastructure with Terraform.

## Getting Started

```bash
git clone <repo-url>
cd <project-folder>
cd <codename>/<learn-step>
cp terraform.tfvars.example terraform.tfvars
# Edit terraform.tfvars with your credentials

# Init terraform
terraform init
terraform plan
terraform apply
# Destroy
terraform destroy
```
